子查询

嵌套在另一个查询中的查询，根据位置不同，分为：where型，from型，exists型。注意：不管子查询在哪里，子查询必须使用()括起来。

where型

where的判断条件是一个查询结果。

select max(math) from score; -- 查找数学的最高分
select id from score where math=(select max(math) from score); -- 查找数学最高分对应的score的id
select name,gender from student where id = (select id from score where math=(select max(math) from score)); -- 查找数学最高分的学生姓名和性别

select * from student where id in (select id from score where math>80);

from型

from的数据源来自一个查询结果，此时这个查询结果必须要起别名。

select name,gender,math,english from (select student.*, score.math,english from student,score where student.id = score.id) as s where math>60 and english>80;

视图

当一个子查询的结果非常复杂，且经常会使用到时，可以考虑将这个查询结果单独保存到一个视图表中。

语法:

create view 视图名 as 查询语句;

create view student_score as select student.*,score.math,english from student join score on student.id=score.id;

删除视图表:

drop view 视图名;

练习:

  1. 查询每个城市中，数学最高的学生信息。

    -- 创建视图表
create view student_score as select student.*,score.math,english from student join score on student.id = score.id;

select * from student_score right join (select city,max(math) as mm from student_score group by city) as s on student_score.city = s.city and student_score.math=s.mm;

  2. 查询每个城市，英语最低的学生。

  3. 计算男生和女生数学和英语成绩的平均分。

    select round(avg(math),2),round(avg(english),2),gender from student join score where student.id = score.id group by gender;

  4. 求岁数最大和最小的学生差多少天

    select DATEDIFF((select max(birthday) from student),(select min(birthday) from student));

-- 查询的源一样，可以使用同一个from语句
select DATEDIFF(max(birthday),min(birthday)) from student;
