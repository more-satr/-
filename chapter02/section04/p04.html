多表查询

我们在查询中，不仅可以查询单一的表格，同时还能联合多个表格进行查询。

多表查询有两种方式:

  • 在from语句中添加多个数据源。
  • union语句拼接多个表格。
  • 使用join...on语句对多个表进行连接。

from的多表查询

在from后面添加多个数据源，也能实现基本的多表查询。这种多表查询的方式不常用，虽然语句简单，但是功能比较有限，能够完全被join语句替代。

select * from student,score;
select * from student join score;
select * from student inner join score;

两表关联，把左表的列和右表的列通过笛卡尔积的形式表达出来。

还可以通过where或者on语句进行条件筛选查询。

select * from student,score where student.id=score.id;
select * from student join score on student.id=score.id;
select * from student inner join score on student.id=score.id;

union语句拼接表格

UNION 操作符用于合并两个或多个 SELECT 语句的结果集。

union要求:

  1. 两边 select 语句的字段数必须一样
  2. 两边可以具有不同数据类型的字段
  3. 字段名默认按照左边的表来设置
  4. 用法:
desc user1;
/*输出
+----------+--------------+------+-----+---------+----------------+
| Field    | Type         | Null | Key | Default | Extra          |
+----------+--------------+------+-----+---------+----------------+
| id       | int(11)      | NO   | PRI | NULL    | auto_increment |
| username | varchar(32)  | YES  |     | NULL    |                |
| password | varchar(512) | YES  |     | NULL    |                |
+----------+--------------+------+-----+---------+----------------+
*/

desc user2;
/*输出
+----------+--------------+------+-----+---------+-------+
| Field    | Type         | Null | Key | Default | Extra |
+----------+--------------+------+-----+---------+-------+
| uid      | varchar(32)  | YES  |     | NULL    |       |
| uname    | varchar(32)  | YES  |     | NULL    |       |
| password | varchar(512) | YES  |     | NULL    |       |
+----------+--------------+------+-----+---------+-------+
*/

select * from user1 union select * from user2;
/*输出
+------+----------+----------+
| id   | username | password |
+------+----------+----------+
| 1    | jack     | 12342355 |
| good | tony     | 324345re |
+------+----------+----------+
*/

使用sql语句删除表格中的部分数据，让student表和score表里的数据不一致。

delete  from score where id=8 or id = 12;
delete from student where id=5 or id=9;

  • Full JOIN全连接

    FULL JOIN 的连接方式是只要左表（table1）和右表（table2）其中一个表中存在匹配，则返回行。相当于结合了 LEFT JOIN 和 RIGHT JOIN 的结果。

    注意，MySQL中不支持Full JOIN，但是可以使用LEFT JOIN,RIGHT JOIN 再配合UNION来实现。

    select * from student left join score on student.id = score.id
union
select * from student right join score on student.id = score.id;
