查询语句

在select语句后面，可以再加其他的语句，这些子语句并不是必须的，但是如果出现的话，一定要按照如下的顺序编写。

  1. from: 从哪些表中筛选
  2. on:多表查询。
  3. where:从表中筛选条件。
  4. group by:分组依据。
  5. having:在统计结果中再次筛选。
  6. order by: 排序。
  7. limit: 分页。

select语句

  • select 既可以做查询, 也可以做输出。

  • select后面添加需要查询的字段，或者使用*表示查询所有的字段。

select 1+1;
select id,name from student;
select * from student;

from字句

from用来指定数据源，可以是一个数据源，也可以是多个数据源。通常情况下数据源是一个表名，但是也可以是一个查询结果。

select * from student; -- 从指定的表里查询所有的字段
select student.name,score.math from student,score where student.id=score.id; -- 从两个表里查询数据
select * from (select student.name,score.math from student,score where student.id=score.id) as student_score where math > 80 ;  -- 从一个查询结果里再次查询

on子句

通常配合join语句来实现多表查询。

select * from student inner join score on student.id=score.id;

where子句

where用来设置查询条件。

  • 示例

      select name from student where city = '上海';
  select * from score where math>=80 and english<=60;

  • 空值判断: is null | is not null

      select `name` from `student` where `description` is null;
  select `name` from `student` where `description` is not null;

  • 范围判断:

    • between ... and ...

    • not between ... and ...

      select id, math from score where math between 60 and 70;
select id, math from score where math not between 60 and 70;

  • 子查询：在select语句里还有一个select语句。

    select * from student where id in (select id from score where english > 60);

group by子句

  • 按照某一字段进行分组, 会把该字段中值相同的归为一组, 将查询的结果分类显示, 方便统计。

  • 如果有 WHERE 要放在 WHERE 的后面

  • 语法: select 字段 from 表名 group by 分组字段;

  • 示例

    select gender, count(id) from student group by gender;
-- 添加where语句
select gender, count(id) from student where city='上海' group by gender;

group by语句需要配合聚合函数使用。详见函数。

having 子句

  • HAVING 和 WHERE的使用功能一样，都是MySQL中的条件查询语句 。
  • 语法: SELECT 字段 FROM 表名 HAVING 条件;
  • 区别在于：
    • WHERE: 后面不能加上聚合函数，只能写在数据源的后面。
    • HAVING: 条件字段必须要在结果集中出现, HAVING 可以写在 GROUP BY 的后面
    • HAVING可以使用字段的别名，但是WHERE不能使用别名。
# 查询每个城市里 money 里的最大值，并且最大值大于 50 的数据
select name,max(money),city from student group by city having max(money) > 50;

select name,max(money),city from student group by city where max(money) > 50;  -- 报错，where 后面的条件不能是聚合函数 

select id i, name n from student where n='周杰伦';  -- 报错,给查询的字段起别名以后，不能再使用wehre
select id i, name n from student having n='周杰伦';

ORDER BY: 按字段排序

  • ORDER BY 主要作用是排序
  • ORDER BY 写在 GROUPBY 后面 ,如果有 HAVING 也要写在 HAVING 的后面
  • 语法: select 字段 from 表名 order by 排序字段 asc|desc;
  • 分为升序 asc 降序 desc, 默认 asc (可以不写)

  • 示例

      select * from student order by age;
  select * from student order by age desc;

LIMIT: 限制取出数量

  • 语法

      select 字段 from 表名 limit m;  -- 从第 1 行到第 m 行
  select 字段 from 表名 limit m, n;  -- 从第 m 行开始，往下取 n 行
  select 字段 from 表名 limit m offset n;  -- 跳过前 n 行, 取后面的 m 行

DISTINCT: 去重

  • 示例

      select distinct city from student;
